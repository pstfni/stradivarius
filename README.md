# stradivarius

A TUI app to browse, filter, and curate tracks from your Spotify playlists into new ones. 


Built with [textual](https://textual.textualize.io/) and [chopin](https://pstfni.github.io/chopin/). 

## Documentation

Full setup and usage instructions at [pstfni.github.io/stradivarius](https://pstfni.github.io/stradivarius).
